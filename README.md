# MAI-NDVW - AI-Enhanced Roguelike RPG

Unity 2D roguelike RPG integrating multiple AI systems for procedural content generation.

## Team

- **Uri** - BSP map layout generation
- **Dani** - Cellular Automata room generation
- **Bruno** - Reinforcement Learning boss AI
- **Jean** - Generative AI (narrative, vision, music)

## AI Systems

- **Narrative Generation** - LLM-based story content
- **Vision Analysis** - Screenshot → room description
- **Music Generation** - Context-aware ambient audio
- **Boss AI** - RL-trained adaptive enemies
